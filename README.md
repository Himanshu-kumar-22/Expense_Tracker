# Expense Tracker – Travel Diary & Group Expense Manager

An all-in-one web application that allows groups (especially travel companions) to track and split expenses while also journaling their trip memories.

> 🛠️ Currently in active development. Stay tuned for new features and updates.

---

## ✨ Key Features (Planned & In Progress)

- 👥 Create and manage travel groups
- 💸 Add, split, and track shared expenses
- 🧾 View balances and settle up between members
- 📓 Travel diary entries with media support
- 📊 Real-time performance and balance summaries
- 🔐 Secure login and authentication

---

## 🧑‍💻 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Auth:** JWT, bcrypt
---

## 📂 Project Status

This project is currently in its development phase. The basic structure and user auth flow are being finalized. Additional features will be released incrementally.
